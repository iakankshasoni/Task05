 Bank Account Simulation - Java OOP

## 📌 Overview
This project simulates basic banking operations using Java’s Object-Oriented Programming (OOP) concepts.  
It includes features such as depositing money, withdrawing money, calculating interest for savings accounts,  
and keeping track of transaction history.

## 🛠 Key Concepts Used
- **Classes and Objects**
- **Inheritance**
- **Method Overriding**
- **Encapsulation**

## 🚀 Features
- Create a bank account with an initial balance.
- Deposit funds into the account.
- Withdraw funds with validation.
- Maintain a transaction history.
- Calculate and add interest for savings accounts.

## 📋 Requirements
- Java Development Kit (JDK) 8 or higher
- Any Java IDE (VS Code, IntelliJ IDEA, Eclipse) or terminal/command prompt
